
## 项目背景



​	蓝沃服饰位于东莞虎门，是一家又多年历史的服饰成衣加工制造企业，传统的销售，跟单，采购，生成，排产和财务回款等，依赖纯手工的方式，效率亟待提升。

​	为提高企业的管理效能和协作效率，针对企业当前情况，落地了数字化转型的蓝沃生产协作系统。

​	经过两周的业务梳理，和实地工厂的考察，针对企业实际的问题，设计和开发了蓝沃生成管理系统。

## 业务范围



系统涵盖的模部分包括：

- 系统权限系统
- 客户管理
- 供应链管理
- 工厂管理
- 样板打样管理
- 生产管理
- 财务应收管理
- 采购应付管理
- ……
